Function

機能概要

メニュー

枠や吹き出しなど、画面上に描画するオブジェクトをメニューに応じて切り替える。 また、利便性を向上させるため、各メニューボタンは、クリックだけでなく、対応するショートカットキーを 押下することでも切り替えることができる。ショートカットキーの詳細は後述。

以下では、各メニューの概要について記述する。

枠

画面上に赤の枠線を描画する。 画面上のクリックされた位置を始点とし、マウスが離された時点での座標をもとに導出された大きさの枠が描画される。 このとき、クリックした位置に既に枠が描画されていた場合、既存のオブジェクトのリサイズが行われる。 リサイズの向きについては、オブジェクトにフォーカスした時点で表示されるカーソルに準じた方向とする。

テキスト

画面上にテキストを描画する。 テキストは、画面上の任意の位置に描画可能であるが、テキストを内包可能な図形の内部で描画する際は、以下の制約が生じる。

  • テキストが描画される始点は、図形の描画始点に準ずる
  • テキストの幅は、図形の幅と連動する

テキストは、画面に描画した後にも編集可能であり、画面上でも視覚的に判断できるよう、カーソルが「テキスト入力モード」へと切り替わる。

吹き出し

画面上に赤枠白塗りの吹き出しを描画する。 吹き出しは、テキストを内包可能な図形であり、枠線と同様、リサイズを可能とする。 いわゆる吹き出しの「しっぽ」部分は、吹き出しの大きさから固定で描画され、「しっぽ」部分単体では編集はできないものとする。

画像

「Ctrl + V」キーが押下されると、クリップボードにコピーされている画像を画面上に描画する。 このとき、画面領域よりも大きなサイズの画像は自動でリサイズされる。また、画像は、システム上では、リサイズ可能な四角として扱われるので、 任意の大きさにリサイズすることも可能である。

移動

画面上に描画されているオブジェクトを移動する。 オブジェクトの付近でマウスをクリックすると、オブジェクトがマウスの位置を追従するようになり、クリックが解除されると、その時点の座標へ 図形が再配置される。また、オブジェクトが移動可能なときは、マウスカーソルが「掴むモード」のアイコンへと変化する。

削除

画面上に描画されているオブジェクトを削除する。 オブジェクトの付近でマウスをクリックすると、オブジェクトが画面上から削除される。 オブジェクトが削除対象となっているかは、マウスカーソルが「掴むモード」のアイコンとなっているか否かで判別する。

セル

本システムは、Excelアプリケーションで直接編集した描画オブジェクトの代替となるものを生成することも、目的としている。 また、Excelアプリケーションでは、セルの大きさは固定ではなく、プロジェクトによって変動し得るものなので、 セルの大きさもプロジェクトに応じて変更できるよう、ユーザの入力値に基づいて描画する。 これは、Excelアプリケーションが、透明色を「黒」と判断し、描画対象が透明なままでは、正常に貼付ができないことによる。

以下では、セルの描画機能について記述する。

セル幅

入力された数値が画面上に描画されるセル幅と対応する。 セルの高さがあわせて入力された段階でバリデーション・描画処理を行う。 いずれかが数値でない場合はセルの描画は行わず、入力値をクリアする。

セル高さ

入力された数値が画面上に描画されるセルの高さと対応する。その他の仕様は、セル幅と同一のものとする。

描画スタック

複数の画像を編集する際、スクロールが伴うと、多くの時間を要してしまう。よって、本システムでは、 「描画スタック」という機構を作成し、スクロールなしで複数の画像を扱えるようにした。 描画スタックは文字通り描画オブジェクトを連ねたスタックを表しており、以下の機能を持つ。

サムネイル

各描画スタックを判別できるよう、それぞれの描画オブジェクトをサムネイルで表示する。

切り替え

描画スタックはそれぞれ割り当てられた番号と紐づいており、それぞれをクリックするだけでなく、対応する「数字キー」によっても切り替えられる。 このとき、どのスタックで現在描画しているかが明確になるよう、アクティブなものは、背景色を変動させるものとした。

クリップボードコピー

画面右下の「Copy」ボタンをクリック、または対応するショートカットキーを押下することで、クリップボードに、 現在の描画スタックのオブジェクトをコピーする。 クリップボードへのコピーは視覚的には成功したか判別できないため、コピー後には、極小時間メッセージとして、コピーが完了した旨を表示する。

ショートカットキー

上記の多くの作業は、マウスのクリックを伴っていては、時間が掛かってしまうため、各々の操作を、対応するショートカットキーを押下することでも可能とした。 ただし、テキストを入力中の間は、入力したテキストによって意図しない機能が発火するおそれがあるため、当該機能を無効化させる。

以下では、各機能と対応するショートカットキーを記述する。また、ショートカットキーと機能の対応は、システム内で、画面左下のアイコンへフォーカスすることでも参照できる。

  • 枠: R(Rectangle 四角)
  • テキスト: T(Text) ※テキスト入力は「Esc」キーでも解除可能とする
  • 吹き出し: W(Word Balloon 吹き出しの英語)
  • 画像: I(Image)
  • 移動: M(Mode)

  • 削除: D(Delete)

  • 描画スタック: 1~9の数字が各スタックと対応

  • 画像をクリップボードにコピー: C(Copy)
