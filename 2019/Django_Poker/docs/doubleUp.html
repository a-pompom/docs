Double Up

DoubleUpアプリ

繰り返し挑戦可能なダブルアップゲームを行う。

画面定義

ダブルアップ画面

ダブルアップ結果画面

処理定義

処理概要

  • 表示されている数値について、比較対象として表れる数値より「大きいか」、「小さいか」を選択
  • 選択が結果と合致していた場合は、勝利、異なっていた場合は、敗北、そして、数値が合致していた場合は、無効とする
  • 勝利・引き分けの場合は、挑戦回数を引き継ぎ、敗北した場合は、挑戦回数をリセット
  • Retryで再挑戦すると、初期状態へと戻す

ダブルアップを行う。 初期状態として、画面には、数値を一つ表示。ユーザは、次に表示される数値が、 比較対象よりも大きいか小さいかを予想し、ボタンで回答を選択。

画面に表示されている数値、ランダムに算出された数値、ユーザの選択によって、 以下のように勝敗判定を行う。

  • (画面上の数値 = ランダムに算出された数値) →無効
  • 数値同士の大小関係比較結果 = ユーザの選択 →勝利
  • 数値同士の大小関係比較結果 ≠ ユーザの選択 →敗北

算出された数値、勝敗判定の結果を画面へ表示し、ゲームを終了とする。 このとき、勝利・引き分けの場合は、挑戦回数を引き継ぎ、敗北した場合は、挑戦回数をリセットする。 また、Retryによって再挑戦を可能とし、その際、状態は、初期状態へ戻す。
